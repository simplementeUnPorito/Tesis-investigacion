#### Los archivos que Elías estaba leyendo

Lo que hace este commit especialmente rico para la bitácora es que nos permite leer los archivos exactos que Elías tenía abiertos y entender qué estaba revisando.

**`psoc_adc.c`** — 51 líneas, módulo de interfaz ISR/main thread:

```c
static volatile int32 g_psoc_adc_isr_counts = 0;
static volatile uint8 g_psoc_adc_isr_ready = 0u;

uint8 psoc_adc_take_isr_sample(int32 *out_counts)
{
    uint8 ok = 0u;
    uint8 saved;
    if (out_counts == (int32 *)0) {
        return 0u;
    }
    saved = CyEnterCriticalSection();
    if (g_psoc_adc_isr_ready) {
        *out_counts = g_psoc_adc_isr_counts;
        g_psoc_adc_isr_ready = 0u;
        ok = 1u;
    }
    CyExitCriticalSection(saved);
    return ok;
}
```

Este patrón `CyEnterCriticalSection / CyExitCriticalSection` es la forma correcta de transferir datos entre el ISR del ADC_DelSig y el main thread en un PSoC 5LP (Cortex-M3 sin FPU, sin RTOS). El ISR escribe `g_psoc_adc_isr_counts` y pone el flag `g_psoc_adc_isr_ready = 1u`; el main loop llama `psoc_adc_take_isr_sample()` que lee atómicamente y limpia el flag. La sección crítica previene que el ISR interrumpa la lectura en medio de la operación.

Que `psoc_adc.c` esté como módulo separado (en vez de estar embebido en `main.c`) indica madurez arquitectónica. El código dejó de ser un monolito.

**`psoc_hw.h`** — 101 líneas, la HAL (hardware abstraction layer):

```c
#if defined(PGAgain_DEFAULT_GAIN)
    #define PSOC_HW_CLASS PSOC_HW_GEO
    #define PSOC_PGA_DEFAULT_CODE PGAgain_DEFAULT_GAIN
#elif defined(PGA_DEFAULT_GAIN)
    #define PSOC_HW_CLASS PSOC_HW_HAMMER
    #define PSOC_PGA_DEFAULT_CODE PGA_DEFAULT_GAIN
#else
    #error "Hardware no reconocido: el TopDesign debe tener PGAgain (GEO) o PGA (HAMMER)."
```

La HAL detecta en compile-time si el PSoC está configurado para geófonos SM-24 (variante GEO, con `PGAgain`) o para el martillo sísmico (variante HAMMER, con `PGA`). Esto permite que el mismo firmware compilado con distintos `TopDesign` funcione en ambas variantes de hardware sin modificar `main.c`.
