#### Los 5 scripts de pruebas teóricas

##### `p01_ondas_de_cuerpo.m` — 722 líneas

El primer script teórico construye intuición visual sobre ondas P y S en un medio elástico infinito homogéneo. El header del script es un modelo de documentación académica:

```matlab
%% p01_ondas_de_cuerpo.m
% PRUEBA TEÓRICA 01 — Ondas de cuerpo en un medio elástico homogéneo
%
% CONCEPTOS CUBIERTOS
%   1. Del material al movimiento: constantes de Lamé → velocidades Vp, Vs
%   2. Ondas P (compresión) vs ondas S (corte): distintas velocidades  
%   3. Propagación de un pulso: qué tan rápido viaja cada tipo de onda
%   4. Carácter NO dispersivo: todas las frecuencias viajan a la misma
%      velocidad (esto cambiará TOTALMENTE cuando veamos medios estratificados)
%   5. Polarización: P paralelo a propagación, S perpendicular
%   6. Por qué Vp > Vs siempre
%   7. Por qué los métodos de ondas superficiales usan Vs y no Vp
%      (insensibilidad de Vp en suelos saturados)
%
% REFERENCIA PRINCIPAL
%   Foti, S. — Cap. 2, Sec. 2.1.3, Ecuaciones 2.11, 2.17, 2.18, 2.19–2.22
```

Este script valida computacionalmente el punto clave del Capítulo 2: en un medio homogéneo, las ondas no son dispersivas. Todas las frecuencias viajan a la misma velocidad. Eso significa que si se grafica la curva de dispersión de un medio homogéneo, es una línea horizontal — la velocidad de Rayleigh es constante en toda la banda de frecuencias. Esto cambia radicalmente en medios estratificados, que es exactamente lo que hace útil el método MASW.

##### `p02_rayleigh_love.m` — 647 líneas

El script más importante de los cinco. Implementa la curva de dispersión teórica de Rayleigh usando la función `MASWaves_theoretical_dispersion_curve()` del submodulo de terceros:

```matlab
% CONCEPTOS CUBIERTOS
%   1. Por qué existe una onda Rayleigh: ecuación secular (Foti Ec. 2.42)
%      como raíz del determinante de la matriz de rigidez dinámica
%   2. VR ≈ 0.92 Vs — la velocidad de Rayleigh en half-space homogéneo
%   3. Movimiento de partículas: elipse retrógrada, componentes Ux y Uz
%   4. Profundidad de penetración (skin depth) ≈ 0.94 λ_R
%   5. Dispersión: en half-space → curva plana. Con capas → curva dispersiva
%      Esto es lo que MASW mide y luego invierte.
%   6. Love: condición de existencia (VS1 < cL < VS2) y curva de dispersión
%
% LIBRERÍA USADA
%   MASWaves_theoretical_dispersion_curve() — dynamic stiffness matrix method
%   (Foti Ec. 2.74: Φ_R = 0, búsqueda de raíces por cambio de signo del det.)
```

El punto `VR ≈ 0.92 Vs` es uno de los resultados más importantes de la teoría de ondas de Rayleigh en un semiespacio elástico homogéneo con coeficiente de Poisson ν = 0.25. Este script lo verifica numéricamente: se define un modelo de suelo con Vs conocido, se calcula la curva de dispersión, y se comprueba que la velocidad de Rayleigh es efectivamente ≈92% de Vs.

##### `p03_sismograma_imagen_dispersion.m` — 543 líneas

Este script implementa el pipeline completo de MASW:

```matlab
% FLUJO COMPLETO:
%   Modelo de suelo (Vs, Vp, ρ, h)
%         ↓
%   Sismograma sintético u(x, t)   ← simulamos aquí la propagación
%         ↓
%   Imagen de dispersión c-f       ← MASWaves_dispersion_imaging()
%         ↓
%   Curva de dispersión c(f)       ← picked del panel c-f
%         ↓
%   Inversión → perfil Vs(z)       ← (ese es el siguiente script)
%
% CONCEPTOS CUBIERTOS
%   1. Wavelet de Ricker: cómo se describe una fuente sísmica
%   2. Sumación modal: u(x,t) = IFFT[ S(ω)·exp(ik(ω)x) ] / √x
%      (Foti Ec. 2.75 — campo de desplazamiento superficial)
%   3. Dispersión visible en el sismograma: cada frecuencia llega a un tiempo
%      diferente porque k(ω) = ω/c(ω) y c(ω) NO es constante
%   4. Decaimiento geométrico: body waves ~ 1/r, Rayleigh ~ 1/√r
%   5. Fase shift method (Park et al. 1998) — cómo MASWaves extrae c-f
```

La wavelet de Ricker es la fuente sísmica estándar en sismología aplicada: un pulso con perfil de segunda derivada de la gaussiana, que modela bien un impacto de martillo o caída de peso. La sumación modal construye el sismograma sintético sumando las contribuciones de cada modo de Rayleigh en cada frecuencia. La imagen de dispersión es el resultado central: el panel c-f donde cada color representa la energía de la señal a una velocidad de fase y frecuencia dadas.
