# Mode Conversion

Cuando una onda sísmica incide sobre una interfaz entre medios elásticos pueden generarse nuevos tipos de ondas.

Este fenómeno se conoce como **conversión de modos**.

---

## Casos principales

- onda **P incidente**
  → ondas reflejadas y transmitidas **P y SV**

- onda **SV incidente**
  → ondas reflejadas y transmitidas **P y SV**

- onda **SH incidente**
  → solo ondas **SH**

---

## Importancia

La conversión de modos es fundamental para comprender:

- [[Surface Waves]]
- propagación en [[Layered Media]]
- formación de [[Rayleigh Waves]]
